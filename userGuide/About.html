DGen - User Guide

ABOUT DGen:

DGen - API based automation utility tool to create new test data / manipulate the test data in T24 Application based on functional testing requirements and that test data can be used respective functional / Regression testing.

How to Use:

Step 1: Set up – Environment Properties

  1. Click on “DGen - Data” Tab.
  2. Below mentioned User Interface components are available under “DGen – Data” tab.
    1. DGen- Data – Tree Folder Structure to view all Input Data Sheets

Step 2: Input Test Data in DGen- Data
